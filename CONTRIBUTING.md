# Contributing

Thanks for helping improve Screeps Bounty Arena.

## Setup

```bash
npm install
npm run check
npm test
```

## Pull request expectations

- Link the issue or bounty task.
- Keep the PR focused.
- Explain what changed and why.
- Include verification output.
- Avoid broad rewrites unless requested.

## Bounty issue format

Good bounty issues include:

1. Goal
2. Context
3. Likely files
4. Acceptance criteria
5. Verification command
6. Non-goals

## Good first tasks

- Add a builder role.
- Add an upgrader role.
- Improve creep body selection.
- Add tests for role assignment.
- Add source assignment so harvesters do not all target the same source.
- Add room memory cleanup.
